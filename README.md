# Quest WebXR Test Scene

This is a minimal WebXR scene for Meta Quest Browser.

## How to test

1. Deploy this folder to an HTTPS static host such as Netlify, GitHub Pages, or Cloudflare Pages.
2. Open the deployed URL inside the Meta Quest Browser.
3. Wait for the page to load.
4. Press the "Enter VR" button.
5. Confirm that the scene opens in immersive VR.

## Important notes

- The Enter VR button is normal.
- The browser requires the user to press a button before starting VR.
- This is not a full game yet. It is only a working VR scene test.
- Do not add movement, combat, or multiplayer until this test works.
